# AGENTS.md

This file provides context about this Obsidian vault for AI agents.

## Vault Overview

Це сховище є детальною базою знань для керування кампанією настільної рольової гри (TTRPG), ймовірно, Dungeons & Dragons (D&D). Воно використовується для систематизації світобудови, включаючи географію (Ферія, Кірос), історію (Таймлайн) та ключових персонажів (NPC, Гравці, Боги).

Основна мета сховища — підтримка та розвиток кількох ігрових кампаній у кастомному фентезійному світі.

## Organization

Сховище має чітку ієрархічну структуру, зосереджену навколо єдиного домену — `D&D1/`. Організація побудована за функціональним принципом, що розділяє ігровий контент, світобудову та адміністративні нотатки.

- **`D&D1/Гра/`**: Містить усі елементи, необхідні для безпосереднього проведення ігор (Боги, Гравці, NPC, Монстри та детальні записи **Кампаній**).
- **`D&D1/Основна Інф/`**: Зберігає глобальний лор світу, включаючи **Країни**, **Карту Ферії**, **Таймлайн** та загальні відомості про **Світ**.
- **`D&D1/Технічне/`**: Містить шаблони для стандартизації нотаток (наприклад, Шаблон НПС, Шаблон Кампанії) та інформацію про використання тегів.

Особливістю є використання папок з позначкою `(Стара_ітерація)`, що свідчить про активне версіонування контенту та архівацію застарілих даних. Нотатки глибоко пов'язані між собою через географічну та тематичну ієрархію (Країни -> Континент -> Місто -> Фракції), що вимагає уваги до контексту при навігації.

## Key Topics

- Керування та відстеження **Кампаній** D&D/TTRPG (нумеровані сесії та спешли)
- Світобудова та лор фентезійного світу **Ферія** (Континент Кірос, Сулкун)
- Ведення реєстру **NPC** (Друзі, Вороги) та **Гравців**
- Опис **Богів**, **Системи** правил та **Легендарних Артефактів**
- **Таймлайн** та історія світу
- Організація та опис **Фракцій** і **Міжнародних торгових гільдій**

## User Preferences

Користувач демонструє високу оцінку **структурованості** та **систематичності** ведення нотаток. Використання папки `Технічне/` та шаблонів вказує на прагнення до послідовності у форматуванні контенту (особливо для NPC та Кампаній).

Очікується, що агент буде використовувати **офіційний, детальний тон**, характерний для опису лору та ігрових правил. Від агента вимагається глибоке розуміння контексту D&D/TTRPG, а також точне посилання на імена власні (Ферія, Кірос, Сулкун).

Оскільки всі папки та більшість файлів написані українською мовою, агент повинен підтримувати спілкування та генерацію контенту **українською мовою**.

## Custom Instructions

- Під час пошуку актуальної інформації завжди пріоритезуйте нотатки, що знаходяться поза папками `(Стара_ітерація)`. Ці старі ітерації слід використовувати лише для архівного контексту або історії змін.
- При генерації нових нотаток про персонажів або сесії, використовуйте існуючі шаблони з папки `Технічне/` для забезпечення послідовності.
- Агент повинен розуміти та коректно використовувати термінологію D&D/TTRPG (Майстер, NPC, Лор, Кампанія).
- Будь-яка нова інформація про світ має бути узгоджена з існуючим лором, особливо з **Таймлайном** та **Картою Ферії**.
